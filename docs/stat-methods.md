# Статистические методы в Research Insights Toolkit

В документе описана математическая база, используемая для расчета значимости.

## 1. Сравнение долей (Проценты)
Для сравнения двух независимых долей используется **z-критерий с объединенной дисперсией (Pooled Z-test)**.

**Формула:**
$$z = \frac{p_1 - p_2}{\sqrt{p(1-p)(\frac{1}{n_1} + \frac{1}{n_2})}}$$

Где:
- $p_1, p_2$ — выборочные доли.
- $n_1, n_2$ — размеры выборок (базы).
- $p$ — объединенная доля: $p = \frac{x_1 + x_2}{n_1 + n_2}$, где $x = p \cdot n$.

## 2. Сравнение средних (Абсолютные числа)
Для сравнения средних используется **t-критерий Уэлча (Welch's t-test)**, который не предполагает равенства дисперсий выборок.

**Формула:**
$$t = \frac{\bar{x}_1 - \bar{x}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}$$

Где:
- $\bar{x}$ — выборочное среднее.
- $s^2$ — выборочная дисперсия.
- $n$ — размер выборки.

## 3. Расчет для NPS (Net Promoter Score)
Инструмент поддерживает расчёт значимости для NPS-строки на основе строк Promoters, Detractors и BASE. Если в таблице присутствует Neutral/Passives, эта строка может обрабатываться как обычная доля, но для NPS-структуры ключевыми support-строками остаются Promoters и Detractors.
- **Promoters**: вес $+1$
- **Passives**: вес $0$
- **Detractors**: вес $-1$

Дисперсия NPS вычисляется по формуле:
$$\sigma^2_{NPS} = (1 - NPS_{index})^2 \cdot w_{prom} + (0 - NPS_{index})^2 \cdot w_{pass} + (-1 - NPS_{index})^2 \cdot w_{detr}$$
*(Где $w$ — доли соответствующих групп, $NPS_{index}$ — значение NPS от -1 до 1)*. Полученные значения подставляются в t-тест Уэлча.

## 4. Обработка малых баз (Small Bases)
Для обеспечения надежности выводов применяется настраиваемый порог малых баз.
- **Порог по умолчанию**: $n < 50$.
- **Логика**: Если база в колонке ниже установленного порога, результаты статистических тестов для этой колонки игнорируются, а к ячейкам применяется специальное форматирование (серая заливка).
- Порог является настраиваемым и может быть изменен пользователем в панели настроек.
