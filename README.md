# Research Insights Toolkit

Excel-first significance testing toolkit for market research and customer insights teams.

## Core Features

### Unified Auto Detection Mode

One button automatically detects and calculates significance for:

- Proportions
- Means + SD
- Means + Variance
- NPS + Promoters / Detractors
- NPS + SD
- NPS + Variance

### Mixed Table Support

Handles real research tables containing multiple metric types in one range:

- percentages
- means
- NPS

with one or multiple shared bases.

### Output Formatting

- Significance letters written directly into cells
- Bold significant values
- Pale green highlights
- Automatic center alignment

### Utility Features

- Clear significance markers
- Metric diagnostics mode
- Automatic project startup in VS Code

## Strategic Goal

Create the most practical spreadsheet significance tool for research managers and insights teams.
