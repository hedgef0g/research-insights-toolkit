Research Insights Toolkit is a private cross-platform analytics add-in for spreadsheet users, starting with Excel and designed for future Google Sheets support.

The product focuses on helping research and consumer insights managers work faster with ready-made tables by automating statistical significance testing, pricing tools, table QA, wave comparisons, and other high-value post-processing tasks.

Current MVP includes:
- Two-point significance testing for proportions (z-test)
- Shared JavaScript calculation core
- Excel task pane wrapper (Office Add-in architecture)

Planned roadmap:
- Expanded significance methods (means, NPS, weighted bases)
- Multi-column comparisons
- PSM (Price Sensitivity Meter)
- Table diagnostics / QA tools
- Tracker wave comparison
- Google Sheets support

Goal: turn spreadsheets into a lightweight insights platform without forcing users into heavy statistical software.
